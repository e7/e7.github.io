# 私有仓库

  1. 配置GOPRIVATE环境变量

  2. 配置git访问方式，在~/.gitconfig下增加相应配置

    [url "git@xxx.yyy.zzz:"]
    insteadOf = https://xxx.yyy.zzz/

  3. 必须存在具有读权限的ssh key
