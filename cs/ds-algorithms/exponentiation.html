    # 快速幂

    # 原理

    计算a^b原始的方式是将a乘b次，时间复杂度为O(n)，实际上不止计算了a^b，还计算了a^2、a^3...a^(b-1)，幂运算本身是可以任意组合的，比如a^b=a^x * a^y * a^(b-x-y)。于是我们将一个大问题拆解为多个小问题，这符合动态规划的思想。a^2=aa，a^3=aa^2，a^4=a^2*a^2，按二进制拆解是比较常见的做法，时间复杂度为O(logn)。

    # 示例

    #! /usr/bin/env python3

def fast_pow(a, b):
    # base就是当前位时的幂，即a^(2^bit)
    base, result = a, 1
    while b > 0:
        # 这里将b按二进制拆解
        if b % 2 == 1:
            result = result * base
        b //= 2
        base *= base
    return result
        
print(fast_pow(2, 5))
