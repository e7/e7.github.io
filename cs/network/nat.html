# NAT类型

# 名词解释

NAT：网络地址转换（Network address translation），即同一子网下的内网节点访问外网机器都会将内网<ip,port>转换为外网的<ip,port>（即wan口），所以目标机器看到的请求都是来自wan口ip，NAT设备会记录该映射关系。

# 锥型

锥型是指内NAT映射关系保持不变的类型，即只要内网<ip,port>不变，那么出去的外网<ip,port>也不会变，即使访问的目标机器变化也不影响，锥形由此得名。

按照限制级别，锥形NAT又有3种子类型（限制级别由低到高）：

  • 全锥形（full cone）

    最宽松的限制，任何目标机器都能使用这条映射通信。

  • ip限制型（restricted cone）

    只有映射中的目标机器才能通信。

  • 端口限制型（port restricted cone）

    只有映射中的目标机器使用映射中端口才能通信。

# 对称型

对称型是指内网<ip,port>和目标机器的<ip,port>其中只要有1个发生变化，都会建立一条映射，而且通信双方必须严格遵守这种映射（通信限制上与端口限制锥形一致），不然包就会被NAT设备丢弃。
